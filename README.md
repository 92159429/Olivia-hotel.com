# Olivia Boutique Hotel — Design System 1.0

This repository contains the Olivia Boutique Hotel front-end redesign system.

## Phase 1 — Layout System

- Unified container system
- 12-column grid
- Consistent spacing scale
- Stable reading widths
- Desktop / tablet / mobile breakpoints
- Section rhythm
- Image aspect-ratio rules
- Layout utility classes

## Phase 2 — Typography System

- Display, title, section, card, lead, body, small, meta, and eyebrow styles
- Responsive type scale through CSS variables
- Prose rhythm utilities
- Multilingual helper classes
- Manual line-break helpers

## Phase 3 — Components and Header System

- Sticky centered-logo header
- Primary navigation below the logo
- Language switcher
- Button styles
- Editorial cards and image cards
- Room card shell
- Footer shell
- Mobile navigation behavior

## Phase 4 — Sections and Hero System

- Hero and editorial hero layouts
- Split editorial sections
- Feature band
- Gallery grid
- Rooms feature and rooms grid
- Cuisine magazine layout
- Contact information layout

## Phase 5 — Animations and Interaction System

- Reveal animation utilities
- Stagger helpers
- Image zoom and hover interactions
- Hero entrance animation
- Reduced-motion support

## Phase 6 — CSS Entrypoint Fix

- `css/style.css` imports the full design system stack
- Animation variable fallbacks added

## Phase 7–11 — Page Implementation

- `index.html`
- `stories.html`
- `rooms.html`
- `cuisine.html`
- `contact.html`

## Phase 12 — SEO, QA, and RWD Polish

- `robots.txt`
- `sitemap.xml`
- Final responsive spacing polish
- Image safety rules
- Card and surface consistency
- Tablet and mobile hero stacking checks
